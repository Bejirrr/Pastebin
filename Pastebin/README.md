## Deploy Vercel

```
git clone https://github.com/hakutakaid/PasteBIN-On-Vercel.git
```
```
npm install -g vercel --verbose
```
```
vercel login
```
```
edit your .env_example
```
```
vercel --prod --yes
```

## Add Env On Vercel (Manual)
```
REDIS_URL
```
```
ADMIN_PIN
```
